# Гра «Міжпланетна подорож» (Interplanetary Journey)
Ласкаво просимо до репозиторію гри «Міжпланетна подорож». У цій грі гравець керує космічним кораблем і вирушає у небезпечну подорож через космос до далекої планети **Kepler-442b.**

## Опис
У процесі гри гравець повинен керувати космічним кораблем та прокладати шлях через космічний простір, уникаючи уламків та інших небезпек. Кожне зіткнення пошкоджує корабель, тому важливо обережно маневрувати та правильно планувати рух. Основна мета — дістатися до планети **Kepler-442b,** отримавши якомога менше пошкоджень під час польоту.
## Особливості гри
* 🛸керування космічним кораблем за допомогою клавіатури
* 🌍подорож у космосі до планети Kepler-442b
* ☄️поява космічних уламків, яких потрібно уникати
* 🎮підрахунок кількості зіткнень під час польоту
* ✨проста і зрозуміла механіка гри
* 🛰️головна мета — дістатися до планети з найменшою кількістю зіткнень
## Керування
W / ↑ — рух вгору  
S / ↓ — рух вниз  
D / → — прискорення  
Esc — повернення в меню
## Технології
У цьому проєкті планується використання наступних технологій:
* мова програмування **C++**
* середовище розробки **Visual Studio Code**
* бібліотеки для роботи з графікою
## Статус проєкту
Проєкт знаходиться на **початковому етапі** розробки.
У майбутньому планується додати основну ігрову механіку, систему перешкод та досягнення кінцевої планети.
