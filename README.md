# interplanetary-journey
Міжпланетна подорож — це проєкт, написаний мовою C++, який є початковою реалізацією майбутньої гри або симулятора космічної подорожі. Даний застосунок створено з метою розробки структури програми та подальшого розширення функціональності.
Програма розроблялася у середовищі Visual Studio Code з використанням компілятора g++. Поточна версія є базовою і демонструє роботу виконуваного файлу. У майбутньому планується додавання нових можливостей та розширення логіки взаємодії з користувачем.
Для запуску потрібно перейти до розділу Releases, завантажити архів з програмою, розархівувати його та запустити файл interplanetary-journey.exe.
Автор проєкту — Анна Руденко. Поточна версія: v1.0.0.
