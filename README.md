# 🚀 Гра «Міжпланетна подорож» (Interplanetary Journey)

> 2D космічна гра на C++ та SFML про небезпечну подорож крізь астероїдний простір.

---

# 🌌 Про гру

**Interplanetary Journey** — це навчальний ігровий проєкт, створений на C++ з використанням бібліотеки SFML.

Гравець керує космічним кораблем, уникає астероїдів, контролює прискорення двигуна та намагається вижити якомога довше під час міжпланетної подорожі.

Проєкт створюється з акцентом на:

- архітектуру гри
- роботу з графікою
- collision detection
- систему анімацій
- побудову ігрових систем на C++

# Архітектура проєкту

Проєкт використовує object-oriented підхід.

Основні класи:

| Клас | Призначення |
|---|---|
| `Game` | основний game loop |
| `Menu` | меню гри |
| `Rocket` | логіка ракети |
| `Asteroid` | логіка астероїдів |
| `Explosion` | система анімації вибухів |

---

# Збірка проєкту

Компіляція через g++

```bash
g++ src/*.cpp -Iinclude -LSFML/lib -lsfml-graphics -lsfml-window -lsfml-system -o game.exe
```
